# AI English Learning Assistant

AI-powered English learning tools and lesson workflows for Indian learners.

## Features

- Spoken English lesson generation
- Hindi-to-English learning workflows
- AI conversation prompts
- WhatsApp learning automation ideas
- Vocabulary and grammar practice tools

## Goals

This project explores how large language models can help Indian learners improve spoken English using simple conversational methods and automation.

## Planned Modules

- Lesson generator
- Vocabulary builder
- AI chat practice
- Pronunciation guidance
- Daily speaking exercises

## Tech Ideas

- Python
- OpenAI APIs
- Claude APIs
- WhatsApp integrations
- Prompt engineering
